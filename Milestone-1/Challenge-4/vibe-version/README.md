# Vibe Version

This folder contains the task manager app built using a vibe coding tool.

**Tool used:** v0
**Time to generate:** 10 mins
**Prompt used:** Build a simple React task manager app with the following features:

- Input field to add a task (press Enter to add)
- Display list of tasks
- Each task has a checkbox to mark complete/incomplete
- Filter buttons: All, Active, Completed
- Clean UI with basic styling (dark mode preferred)

Technical requirements:
- Use React functional components
- Use useState for state management
- Keep code simple and readable
- Separate components if needed (App, TaskList, TaskItem, Filter)

Return full working code.
