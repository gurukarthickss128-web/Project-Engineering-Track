# CRUD App

## What It Is
A simple CRUD app to manage items.

## The Problem It Solves
I needed a basic system to store and manage items easily.
This app allows creating, viewing, updating, and deleting items.

## What I Intentionally Excluded
- Authentication: Keeps app simple for single-user use.

## Tech Stack
- Backend: Node.js + Express
- Frontend: Vanilla JS
- Deployment: Render + Github Pages

## How to Run
cd backend
npm install express cors dotenv
node server.js
