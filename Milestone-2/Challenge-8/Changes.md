# FridgePolice Prototype

## Overview
A roommate food tracking system that handles real-world inconsistencies.

## Scenario 1: Double Allocation
Handled using validation against approved portions.

## Scenario 2: Expired Approvals
Requests expire after time limit.

## Scenario 3: Identical Items
Each item has UUID.

## Scenario 4: Inventory Correction
Manual adjustment invalidates requests.
